## 项目简介


电商管理系统，使用vue+elementUI，实现的功能有
* 用户管理
* 权限管理
* 分类管理
* 参数管理
* 商品管理
* 数据统计
仓库内容包含【源代码+个人总结笔记+参考资料】

## 如何使用该项目
### 安装依赖
```
npm install
```
### 开发时用到的编译和热加载（启动项目）
```
npm run serve
```
### 生成生产版本（这里生成的dist文件可以部署到服务器上来直接访问）
```
npm run build
```
